This API uses Identity V Fandom's pages to scrape and find data for an item.

Query Parameters

/skin/:name - The name of the skin you want to get its costume link.

/accessory/:name - Get data for an accessory. The :name is the query name for the acessory.

/aliases - Get the aliases for all character names.

/portrait/:series/:name - Get data for a portrait. The :series is the query for the series type. See possible series.
The :name is the query name for the character you want to retreive its series' portrait.

Error Codes

  • 404 - Page Not Found. The API failed to find the fandompage you asked for.
  • 400 - Bad Request. Thrown when a parameter is missing or has wrong value.

Possible portrait series

Portrait series are non-speacial portraits, they're all summed up in one page in the Identity V Fandom, see the colorful memory series' page for reference.

  • Colorful Memory
  • Black and White

Note that the API is flexible with the :series query, meaning that you don't have to write it the way the fandom requires you to.
For instance, using "Black_and_white" is enough instead of "Black-and-White_(Portrait)"
